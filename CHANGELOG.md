# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-21

### Added

- `globMatch()` — test a string against a glob pattern
- `globFilter()` — filter an array of strings by a glob pattern
- `globToRegex()` — convert a glob pattern to a RegExp
- Supports `*`, `**`, `?`, `{a,b}`, and `[abc]` syntax
